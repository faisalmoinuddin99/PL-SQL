SQL COMMANDS :

1. DDL ( Data Definition Language ): CREATE, ALTER, DROP, TRUNCATE these commands mainly deals with the STRUCTURE Of Objects.

2, DML ( Data Manipulation Language ): INSERT, UPDATE, DELETE these commands mainly deals with DATA which stored in a table.

3. DRL ( Data Retrieval Language ): SELECT [ Logical Copy of Data ]

4. TCL ( Transaction Control Language): COMMIT, ROLLBACK, SAVEPOINT
